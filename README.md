# Train Departure Board

Real-time departure dashboard for London stations.
Shows next departures from Wandsworth Town (National Rail) and East Putney (District Line).

## Setup

1. Clone the repo
2. Create virtual environment: `python -m venv .venv`
3. Activate: `source .venv/bin/activate` (Mac/Linux) or `.venv\Scripts\Activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Copy `.env.example` to `.env` and add your API keys
6. Run: `streamlit run app.py`
